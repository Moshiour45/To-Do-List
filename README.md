# To-Do-List# To-Do-List
[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/Moshiour45/To-Do-List)

This project is a clean, modern, and responsive To-Do List application. It provides a straightforward user interface for managing daily tasks. Built with vanilla JavaScript, HTML, and styled with Tailwind CSS, this application is lightweight and easy to use.

## Features

*   **Add Tasks:** Quickly add new tasks to your list using the input field.
*   **Delete Tasks:** Remove individual tasks with a single click.
*   **Clear All:** Instantly clear the entire list of tasks.
*   **Task Counter:** A dynamic counter keeps track of the number of remaining tasks.
*   **Keyboard Support:** Add tasks conveniently by pressing the 'Enter' key.
*   **Responsive Design:** A clean layout that adapts to various screen sizes.

## Technology Stack

*   **Frontend:** HTML, Vanilla JavaScript
*   **Styling:** Tailwind CSS

## Getting Started

Follow these instructions to get a local copy of the project up and running.

### Prerequisites

You will need [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) (which comes with Node.js) installed on your system.

### Installation and Setup

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/moshiour45/to-do-list.git
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd to-do-list
    ```

3.  **Install dependencies:**
    This will install Tailwind CSS and its CLI tool.
    ```sh
    npm install
    ```

4.  **Build the CSS:**
    The project uses Tailwind CSS for styling. Run the following command to compile the `src/input.css` file into `output.css`. The `--watch` flag will automatically recompile the CSS when you make changes to the input file or your Tailwind configuration.
    ```sh
    npm run build
    ```

5.  **Run the application:**
    Open the `index.html` file in your preferred web browser to view and interact with the application.
